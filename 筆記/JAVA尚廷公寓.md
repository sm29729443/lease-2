# JAVA 尚廷公寓

## 學習目的

此次為第二次學習此項目，第一次學習時並沒有太了解項目的業務邏輯，以及資料表的結構，使得在實作 api 時，儘管只是簡單的 CRUD，但有些部分會因為對資料表的不熟悉，而聽得不是很明白，所以此次會詳細理解資料表的結構是如何設計的。

### 主要學習點

1. 了解一般前後端項目開發流程
2. 了解資料庫的具體設計步驟
3. 複習 MyBatis-Plus 使用方式
4. 了解 redis 的具體應用場景
5. 了解 nginx 的部屬流程

## 項目概述

此項目為一個公寓租賃平台，又分為移動端(app)、後台管理端。

### 移動端

移動端就是一般用戶使用的介面，這裡設計成 mobile 的介面。
